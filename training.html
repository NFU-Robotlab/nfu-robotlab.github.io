📌 新手訓練指南說明

歡迎加入 智能與機器人系統實驗室 (IRS Lab)！本清單整理了進入實驗室進行研究與專案開發前所需具備的基礎能力。建議依序完成下列學習目標，勾選框會自動記錄你的學習進度。

📐 1. Onshape 3D建模

Onshape 是全雲端架構的 3D CAD 建模軟體，免安裝且支援多人即時協作，是實驗室進行機器人機構設計、連桿干涉驗證以及 3D 列印輸出的主要工具。

🎬 前往 Onshape 基礎教學
ℹ️ 點擊上方按鈕前往實驗室製作的 Onshape 基礎教學課程。

🔌 2. 實驗室基本設備

學習雷射切割機與 3D 列印機的使用方式，以利後續進行專案機構件、機器人外殼與結構零件的實體加工製作。

⚡ 雷射切割機操作與安全守則 🖨️ 3D 列印機操作與切片軟體設定
ℹ️ 點擊上方按鈕，可以直接跳轉至自製設備教學頁面的對應主題。

🤖 3. Robotis 套件

學習使用 Robotis 軟硬體套件與 CM-530 控制器，控制 AX-12(A) 智慧型序列伺服馬達，並掌握關節與輪子模式控制，實現實體機器人的機構作動。

⚙️ 前往 Robotis 基礎教學
ℹ️ 點擊上方按鈕進入實驗室的 Robotis 控制器與馬達程式撰寫教學。

💻 4. C# 程式

學習使用 C# 語言與 Visual Studio 開發視窗程式，透過 SerialPort 序列通訊埠發送 Protocol 1.0 指令封包，實現電腦端對智慧伺服馬達的精準雙向控制。

⚡ 前往 C# 程式與通訊教學
ℹ️ 點擊上方按鈕進入 C# 專案建立、封包結構與機械手臂控制驗收指引。
